# AI Enabled College Assistant System 🎓🤖

An AI-powered college assistant website developed to help students access important college-related information in a simple and organized way.  
The project provides guidance related to admissions, courses, placements, and career options through an interactive web interface.

This project was created for learning and academic purposes.

##⚙️ Working of the System
Users open the website and access the welcome page.


Students can explore college-related questions and answers.

The system provides organized information about courses, admissions, and placements.

Users can navigate through different sections easily.

Career guidance options help students explore different career paths.
